OMDB: Object-Rational Database Design

Object-Relational Database - 31075 - UTS
[PDF]   [CODE]

This project involves the design and implementation of an Online Movies Database (OMDB). OMDB maintains information about movies being screened in cinemas near users. The database supports a website that allows the users to query this information and search for movies and cinema sessions. Each movie is screened in one or more cinemas. The event listings show cinema sessions for each movie for one week (i.e. movies screened at a given cinema for seven days, starting today).
